# 🚀 Developer Personal Portfolio

A modern, fully responsive and interactive Developer Portfolio Website built using pure HTML, CSS, and JavaScript. This portfolio showcases skills, projects, education, achievements, experience, and contact information with a beautiful Glassmorphism UI and smooth animations.

## ✨ Features

* Modern Glassmorphism Design
* Fully Responsive Layout
* Dark / Light Mode Toggle
* Animated Typing Effect
* Interactive Skills Section
* Project Showcase Cards
* Education Timeline
* Experience Timeline
* Smooth Scroll Animations
* Animated Counters
* Mobile Friendly Navigation
* Contact Form UI
* Floating Particle Background
* Clean and Professional UI

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📂 Sections

* Home
* About
* Skills
* Projects
* Education
* Experience
* Achievements
* Contact

This project was created to showcase my web development skills, projects, achievements, and professional journey in a modern and visually appealing way.
